This C program places N queens on a chess board in a manner such that none of the queens are capable of attacking each other.
		   
The user is prompted to place a queen in the first row at any column they desire. The remaining N-1 queens are then placed in each following row automatically. The user is then asked if they would like to continue. If the user enters an invalid number, then an error occurs. The current value for N is 8 (BOARDSIZE). It can be edited on line 32 of the code. An output file must be provided in order to run this program. A sample output file is provided.

This program was written for the Advanced C Programming course with UCSC Extension.
